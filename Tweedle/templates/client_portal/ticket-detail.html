Description

When exporting reports to CSV, multiple columns are missing from the output. Specifically, the "Assigned Date" and "Resolved Date" columns do not appear in the downloaded file even though they are visible in the report table on screen. This affects all report exports generated after 14 Apr 2025. Earlier exports (before this date) appear to be fine. Steps to reproduce: 1. Navigate to Reports → Generate Report with any date range. 2. Click "Download CSV". 3. Open the downloaded file — "Assigned Date" and "Resolved Date" columns are absent. Expected: all visible columns should export to CSV. Actual: only Ticket ID, Client, Subject, Priority, Status, and Issue Date columns are present.
